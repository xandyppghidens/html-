    A plataforma Gesto+ foi criada desenvolvida como projeto integrador pela turma 028.2025.0108 – Técnico em Informática do Senac MG, unidade de Araxá. O objetivo é tornar a Libras mais inclusiva como uma linguagem informal e cotidiana através de expressões rápidas como Gifs e figurinhas compartilháveis. O site foi feito para entregar uma linguagem mais rápida o tema escolhido pela professora responsável pelo gerenciamento do projeto integrador.
